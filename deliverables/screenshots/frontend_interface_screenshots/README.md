### frontend_dashboard_overview.png
The main Dashboard page with 4 summary metric cards at the top (Total Interactions, Avg Response time, Error Rate at 0.0%, Services Up showing 4/4). ## Chat

### frontend_chat_empty_state.png
The Nexus Chat page before any messages are sent. Shows the header "Nexus Chat — Multi-agent / Llama 3 8B / Routes to specialized agents" and centered empty state with the Nexus logo, "Ask anything. Nexus routes your message to the right agent." text, and 4 suggestion chips: "What can you do?", "Summarize my docs", "Analyze a review", "Show my digest".

### frontend_chat_general_conversation.png
A general chat conversation showing the Orchestrator routing to the default Nexus agent. User says "hi" and gets a greeting mentioning DocMentor and FileFinder. User asks "what can you help me with" and receives a detailed list of capabilities (answering questions, generating ideas, summarizing content, providing definitions) with a note about routing to specialized agents.

### frontend_chat_docmentor_routing.png
Demonstrates automatic agent routing — user asks "Where can I get some help with documents" and the response is tagged "DOCMENTOR" (green tag). DocMentor introduces itself and lists uploaded source documents by name (test-quick.txt, artificial-intelligence-foundations.txt, data-engineering-mlops.txt, cloud-computing-architecture.txt, biotechnology-genomics.txt).

### frontend_chat_review_analyst_routing.png
Shows the Review Analyst agent being routed to via chat. User asks for feedback on a review and the response is tagged "REVIEW ANALYST" (olive tag). Two review analyses are shown: a positive customer service review (90% confidence, 5 stars, themes: exceptional service) and a negative bar review (80% confidence, 2 stars, themes: high prices, poor service) with extracted pros and cons.

### frontend_chat_system_stats_response.png
User asks "how's is our system looking you got any stats?" and the Nexus agent responds with platform performance statistics: DocMentor 95% accuracy, Review Analyzer 90% sentiment identification, Content Digests averaging 3 minutes generation time, and 99.9% file processing success rate.
